# CSS网页编程语言

### 概念

~~~java
	就是说这个语言也是用来开发网页的，但不完全开发，它属于是控制网页 内容 表现的语言。
    往往跟HTML混合使用，与HTML，JavaScript并称前端三剑客。
~~~

### 常用元素属性

~~~java
字体属性
font-family			字体类型	
font-size			字体大小	
font-weight			字体粗细	
font-style			文本样式	
color				字体颜色	
    
    
文本属性
center				居中对齐
left				左对齐
right				右对齐
underline			下划线 （常用）
none				可以给 a 标签去掉下划线
overline			上划线（不常用）
line-through		删除线 （不常用）

    
背景属性
repeat				平铺
no-repeat			不平铺
repeat-x			水平平铺
repeat-y			垂直平铺
~~~
